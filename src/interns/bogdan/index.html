During an internship I would like to learn by working with professional engineers, because learning by doing is what works for me. I want to experience the sleepless nights trying to figure out how a company’s stack works and get good at it and writing code that I can feel proud of. I want to work in a team of fellow engineers, get to attend real Scrum meetings and possibly make some friends and connections.

I personally like web development, back-end in particular. I want to become a great full-stack software engineer one day so I can make any web software I want. I have had many side projects abandoned due to lack of time needed to learn some back-end knowledge. If I get to work at your company I will be very happy and will contribute with all my enthusiasm for software development.
